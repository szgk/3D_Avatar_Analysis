# About

配布・販売されているオリジナルアバターの一覧と統計

### やっていること・やる事

- 無償公開されている公式情報の分析、統計
- 分析結果の無償公開

### やっていない事・やらない事

- 版権物を使った生成 AI での画像・3D モデル生成、統計の為以外でのデータ生成
- 生成 AI に使うことが目的の学習
- 無断転載画像の利用
- 分析結果の売買
